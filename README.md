🎓 KBTU CodeHelp — Group Project
This project was developed by our team for the Web Development course. It is a platform where KBTU students can ask programming questions (similar to StackOverflow), and other students or Teaching Assistants (TAs) can provide answers.

💻 About the Project (Features):
Categories: Questions are organized by programming languages (Python, C++, Java, JS, etc.).

Tickets (Questions): Students can create a question, describe the problem, and attach code snippets.

Answers & Solutions: Comments can be added to every question. A specific answer can be marked as the "Solution."

API: The entire backend is powered by Django REST Framework.
